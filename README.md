# Python Chatbot - Curso Datalumina

Chatbot inteligente que ensena Python basado en el curso de [python.datalumina.com](https://python.datalumina.com).

## Caracteristicas

- **19 temas** del curso de Python
- **Busqueda inteligente** en espanol natural
- **Ejemplos de codigo** para cada concepto
- **Quiz interactivo** para practicar
- **Temas relacionados** sugeridos

## Temas cubiertos

| Categoria | Temas |
|-----------|-------|
| Funciones | Definir funciones, Parametros, Valores de retorno, Alcance |
| Variables | Variables, Numeros, Strings, Booleans |
| Estructuras | Listas, Diccionarios, Tuplas, Conjuntos |
| Control | If statements, Loops |
| POO | Clases |
| Errores | Try/Except |
| Otros | Organizar codigo, Primer programa, Por que Python |

## Requisitos

- Python 3.6 o superior

## Uso

```bash
python python_chatbot.py
```

## Comandos

| Comando | Descripcion |
|---------|-------------|
| `/ayuda` | Muestra ayuda |
| `/temas` | Lista todos los temas |
| `/quiz` | Quiz aleatorio |
| `/salir` | Sale del chatbot |

## Ejemplos de preguntas

```
Que son las funciones?
Como creo una lista?
Enseñame if statements
Que es un diccionario?
Dame un ejemplo de loops
Como manejo errores en Python?
```
