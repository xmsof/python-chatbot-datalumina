# Monica AI - Chatbot Inteligente

Asistente inteligente que responde CUALQUIER pregunta.

## Como usar

### Windows
1. Doble clic en `instalar.bat`
2. Doble clic en `iniciar.bat`

### Mac/Linux
```bash
pip install -r requirements.txt
python python_chatbot_gui.py
```

## Ejemplos

- Que es una pizza?
- Que es bitcoin?
- Como funciona el internet?
- Contame un chiste
- Receta de tacos al pastor

## Tecnologias

- Python 3.6+
- Groq API (Llama 3.1 8B)
- Wikipedia + DuckDuckGo
- Tkinter (GUI)
